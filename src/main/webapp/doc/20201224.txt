maven clean: 메이븐 빌드과정에서 생성된 결과물(target) 삭제
mvn package -> class 컴파일, war 파일로 생성 
                          artifacted-version.war 			
			

maven lifecycle - 개념적 이해 -> 단계 순서 정형화함 
clean: 기존 build 작업 내역 삭제 , default lifecycle - 우리는 package함, site lifecycle
